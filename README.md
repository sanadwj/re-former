##  Re-Form 

In this project we built some forms, both using nearly-pure HTML and then graduating to using the helper methods that Rails provides.


## Build With

- Ruby 2.7.0p0
- Ruby on Rails 5.2.4.4

### Setup
 - Open a terminal
 - Copy this code : 
        ```
        git clone git@github.com:sanadwj/re-former.git
        ```
